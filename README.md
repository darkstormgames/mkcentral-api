# mkcentral-api
A simple API wrapper for the Mario Kart Central website for NodeJS.

## Installation

Node.js 16.0.0 or newer is required.

```
npm install mkcentral-api
```

## Usage 

### JavaScript

```js
const MKC = require('')
```
